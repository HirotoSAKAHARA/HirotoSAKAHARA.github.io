サポートベクトルマシン(基本)

英語名、略称など

SVN、Support Vector Machine。分類に使われる場合はSupport Vector Classification(SVC)とも。

ターゲットタスク

分類、回帰、クラスタリングなど、ほぼあらゆる機械学習に使われる。

概要

多次元空間上のデータを超平面（サポートベクトル）により分類する手法。

長所

  • 局所解収束の問題がない。
  • 逆誤差伝播法などと違い、できるだけマージンを取って分類するため、未学習データに強い。

短所

  • 多クラス分類などでは計算量が多くなる。
  • 非線形分類に対しての設計基準がない。

次元削減

PCA : principal component analysis, 主成分分析

主成分分析を行い、不要な次元を削減することで、学習パラメータの大幅削減を行う。

SVC : singular value decomposition, 特異値分解

特異値分解を行い、不要な次元を削減することで、学習パラメータの大幅削減を行う。 特異値分解とは、行列\(\bm{A}\)を\( \bm{A} = \bm{U}\sum \bm{V}^{\top} \)と分解することである。ここで\(\sum\)の次元は\(\bm{A}\)と等しく、\(\bm{D}\)を\(\bm{A}\)の固有値の対角行列とすると、\( \bm{\sum} = \left( \begin{array}{cc} \bm{D} & 0 \\ 0 & 0 \end{array} \right) \)と表される(0の数で次元を合わせる）。\(\bm{V} \)は特異ベクトルを並べた右特異ベクトル、\(\bm{U}\)は\(\bm{A}\)、\(\bm{\sum}\)、\(\bm{V}\)から求まる左特異ベクトルである。
